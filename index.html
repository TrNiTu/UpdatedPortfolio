Computer Systems Engineering

Course Code: CSCE231

Course Grade: A

Computer Systems Engineering is a course that is more hardware focused rather than software focused; however it was centered on how computers work. For an example, one of our projects was to read and write assembly in order to defuse a fake bomb as well as using injections to "hack". The final project in this course was to use an Altera motherboard and have the switches on it do different things when turned on and off, for an example having a light turn on when the switch is on. The primary languages used in this course is C and x86 Assembly.
